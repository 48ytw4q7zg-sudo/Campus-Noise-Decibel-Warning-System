## 五、本轮优化思路

本轮以"修复第1轮发现的最高优先级问题"为核心目标，优先保证安全性和正确性：
1. **安全第一**: JWT秘钥环境变量化消除最严重的安全漏洞
2. **逻辑正确**: 修复前端 isAbnormal 比较类型错误，确保异常状态标签正确渲染
3. **数据防护**: CSV导出公式注入防护，避免用户导出数据后在Excel中触发恶意公式
4. **性能优化**: 仪表盘 N+1 查询优化为批量查询，4 次循环 DB 调用 → 3 次批量查询
5. **模型完善**: 混合模型性能从硬编码改为基于实时 DB 数据统计计算

剩余问题均为 P2 级别（测试扩展、非核心功能优化、教学简化声明范围内的限制），不阻塞系统上线。
